The Experiment: Lucid Capture

Built an interactive product demo tool (like Arcade.software) using Stitch as the design source of truth.

  • Chrome extension captures clicks on any website
  • Editor: hotspots, annotations, timing, drag-and-drop
  • Player: full-screen immersive playback
  • Export: single self-contained HTML file
