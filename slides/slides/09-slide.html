Design Tokens → Tailwind Theme

Stitch exports structured JSON with 48 named colors, fonts, spacing:

{
  "primary": "#0029c0",
  "surface": "#f9f9ff",
  "on_background": "#141b2b",
  "surface_container_high": "#e1e8fd"
  ...
}

Drives the Tailwind @theme block in both React and Go stacks:

@theme {
  --color-primary: #0029c0;
  --color-surface: #f9f9ff;
}

One source of truth → both stacks use identical colors.

Talking Points (1.5 min)

This is Stitch's strongest feature. The tokens are structured, diffable, and directly usable. No manual color picking. When the tokens change, git diff shows exactly what shifted.
