VIDEO or LIVE DEMO (2 min)

This video shows: opening Stitch, the project with screens, clicking through design tokens, viewing HTML output. Pre-recorded so you don't wait for page loads.

To record this video:

  1. Open QuickTime → File → New Screen Recording
  2. Go to stitch.withgoogle.com
  3. Open the project, click through screens, show design system panel
  4. Show the HTML output for one screen
  5. Save as videos/StartingStitch.mov (or .mov)

If doing live instead:

Skip the video, switch to browser, walk through Stitch UI live.

Talking points while video plays:

  • "This is Stitch — Google's AI design tool from Labs"
  • "Notice the design system panel — structured tokens, not just colors"
  • "Each screen uses the same tokens, fonts, spacing"
  • "The HTML output uses Tailwind classes — familiar territory"
