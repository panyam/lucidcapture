Model Context Protocol — Stitch exposes its entire API as MCP tools. Any AI coding assistant can read and write designs programmatically.

MCP ToolWhat It Does
get_projectRead design system, tokens, fonts
list_screens / get_screenList and download screen HTML
generate_screen_from_textCreate new screens from a prompt
generate_variantsLayout, color, font variants of existing screens
edit_screensPush changes back to the design
