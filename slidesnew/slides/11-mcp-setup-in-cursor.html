MCP Setup in Cursor

🔴 LIVE DEMO

// .cursor/mcp.json
{
  "mcpServers": {
    "stitch": {
      "url": "https://stitch.googleapis.com/mcp",
      "headers": {
        "x-goog-api-key": "$VIBESTITCH_API_KEY"
      }
    }
  }
}

One config file. Restart Cursor. Stitch tools appear in the AI chat.

DEMO STEPS (2 min)

BEFORE THE DEMO — set up the project:

cd ~/work/misc/stitchdemo
./setup.sh                    # prompts for project ID + API key
# OR:
./setup.sh PROJECT_ID API_KEY  # non-interactive
source .env.local              # load API key
./stitch-sync/sync.sh all      # pull initial designs

To change project mid-demo:

echo "NEW_PROJECT_ID" > stitch-sync/project.conf
./stitch-sync/sync.sh all

During demo:

  1. Open Cursor, show .cursor/mcp.json
  2. In Cursor chat: "list my Stitch screens"
  3. Show the tools appearing: get_project, list_screens, etc.
  4. "One config file. No SDK, no npm install."
