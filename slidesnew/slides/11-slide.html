MCP Setup in Cursor

🔴 LIVE DEMO

// .cursor/mcp.json
{
  "mcpServers": {
    "stitch": {
      "url": "https://stitch.googleapis.com/mcp",
      "headers": {
        "x-goog-api-key": "$VIBESTITCH_API_KEY"
      }
    }
  }
}

One config file. Restart Cursor. Stitch tools appear in the AI chat.

DEMO STEPS (2 min)

  1. Open Cursor settings → MCP servers
  2. Show the config above (or paste it)
  3. Restart Cursor
  4. In Cursor chat, type: "list my Stitch screens"
  5. Show the tools appearing: get_project, list_screens, etc.
  6. "That's it. No SDK, no npm install, no API wrapper. Just a URL."
