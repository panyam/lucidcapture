First Try: Prompting for Each Sync

🔴 LIVE DEMO (or video)

"With MCP set up, tried the obvious approach — ask Cursor to sync:"

> "Get the design tokens from my Stitch project 
   and save them as design-tokens.json"

> "Download all screen HTML files from Stitch 
   and save to stitch-sync/html/"

> "Extract the structural outline of each screen 
   into diffable JSON"

It works. Each prompt takes 30-60 seconds. For 10 screens that's 10 minutes of waiting.

And you have to do this every time the design changes.

DEMO STEPS (1.5 min)

  1. In Cursor chat: "Get the design tokens from my Stitch project and save as stitch-sync/design-tokens.json"
  2. Wait for it to run the MCP call, process, save...
  3. "See? It works. But it took 45 seconds for ONE file."
  4. "Now imagine doing this for 10 screens, every time something changes."
  5. Pause. "There's a better way."

Alternative: show a pre-recorded video of this prompt running if you don't want to wait live.
