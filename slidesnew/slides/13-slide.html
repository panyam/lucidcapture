Principle #1

"Don't prompt an LLM to solve a problem when you can ask it to write a tool."

🐌

Prompting

45 sec per file
Manual every time
Non-deterministic

→

⚡

Tool

2 sec for everything
One command
Deterministic, repeatable

Key moment (1 min)

"The LLM is great at writing the sync script. It's terrible at BEING the sync script. Ask it once to write the tool, then use the tool forever."

"This is a general principle for working with AI coding assistants — don't repeat prompts. Extract patterns into scripts, skills, commands."

Then transition: "So I asked Cursor to write a sync script..."
