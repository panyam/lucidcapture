DEMO STEPS (1.5 min)

  1. Open terminal in the project
  2. Run: ./stitch-sync/sync.sh all
  3. Show the output — manifest, HTML files, screenshots, structure
  4. Run: ls stitch-sync/ — show the directory structure
  5. Run: cat stitch-sync/design-tokens.json | head -20
  6. "This JSON drives the Tailwind @theme block. One source of truth."
  7. Run: git diff stitch-sync/ — "every change is trackable"
