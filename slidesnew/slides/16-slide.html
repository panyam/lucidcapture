Tokens → Tailwind Theme

// stitch-sync/design-tokens.json (from Stitch)
{ "primary": "#0029c0", "surface": "#f9f9ff", 
  "on_background": "#141b2b" ... }

// app/src/index.css (drives both React + Go stacks)
@theme {
  --color-primary: #0029c0;
  --color-surface: #f9f9ff;
  --color-on-background: #141b2b;
}

// Dark mode — one CSS override, everything switches
.dark {
  --color-surface: #0f1219;
  --color-on-background: #e2e4f0;
}

48 named colors. One source of truth. Dark mode for free.

Talking point (1 min)

Show how the tokens flow from Stitch JSON to CSS custom properties. "Every element using bg-surface or text-on-background automatically switches in dark mode. No dark: classes needed."
