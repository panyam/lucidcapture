Principle #2

"Build skills, not prompts."

/stitch-sync READ-ONLY

Pull designs from Stitch, show git diff of what changed. Runs sync.sh under the hood — deterministic, no LLM in the loop.

Undo: git checkout stitch-sync/

/stitch-diff READ-ONLY

Structural diff — compares JSON outlines, not raw HTML. Shows new sections, removed elements, text changes. Ignores Tailwind class noise.

No side effects — analysis only

/stitch-variants WRITES TO STITCH

Generate layout, color, or font variants via MCP. Creates new screens in Stitch (async — 2-5 min). Run /stitch-sync after to pull results.

Undo: tombstone the unwanted screen IDs

/stitch-analyse READ-ONLY

Compare Stitch HTML against your templates. Produces a section mapping table — what maps, what's new, what differs. Porting checklist for implementation.

No side effects — analysis only

Each command is a markdown file in .cursor/commands/ — checked into git, shared with the team.
