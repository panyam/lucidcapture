DEMO STEPS (3 min)

  1. Switch to Stitch browser tab — generate an About screen using the Stitch UI (text prompt in Stitch, not MCP)
  2. "This time the design starts in Stitch, not in code. Let's bring it down."
  3. Terminal: ./stitch-sync/sync.sh all — show it downloading the new screen
  4. Terminal: ./stitch-sync/sync.sh status — "About" appears in the list ✓
  5. Cursor: /stitch-analyse — show the structural outline of the new screen
  6. Cursor: "Convert stitch-sync/html/about.html into a React component at src/pages/About.tsx using our design tokens. Add a route at /about"
  7. Browser: show /about page with live reload
  8. Terminal: git add -A && git commit -m "Add About page from Stitch design"
  9. "7 steps. Design to production. Deterministic sync, structural analysis, AI for the creative work, git for versioning."
